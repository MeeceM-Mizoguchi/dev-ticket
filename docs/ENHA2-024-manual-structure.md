# ENHA2-024 マニュアル表示機能 ― 詳細構成（ページ設計）確定版

業務フロー順・全8部。各章を「ページ（ステップ）」単位まで分解した設計表。
本マニュアルは**静的なドキュメント**（上から読む形式）。各ステップは「説明文＋コーディング再現した画面（画像貼り付けサイズ感）」で構成し、画面上の対象箇所を**赤枠で囲み、それ以外をグレーアウト**して強調する。アニメーション等の動く演出はなし。

- `表示画面` = `screens/` 配下に作るコーディング再現モック
- `強調箇所` = 対象以外グレーアウト＋対象に赤枠で示す位置（該当なしは「—」）
- `権限` = その操作を使えるユーザー。権限設定なく誰でも使えるものは「全ユーザー」
